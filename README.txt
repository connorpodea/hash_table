My "chaining type" hash table uses SHA 1 as the hash function. 
I created a HashTable class to implement the table and all the required methods. 
The keys are stored in nodes and handled using a pointer array.
At the time of this README, common methods outside of the assignment's requirements are not implemented.
Sha 1 reference: https://nvlpubs.nist.gov/nistpubs/Legacy/FIPS/fipspub180-1.pdf
